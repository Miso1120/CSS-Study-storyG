조리순서

  1. 요즘에는 간편미역이라고 해서 씻은 미역을 20g 잘라담아서 팔고 있습니다. 20g을 다하면 너무 양이 많아서 2인분용으로 절반만 썼어요.
    여기에 물을 넣으면 금방 불어납니다
  2. 키친타월로 소고기 피를 살짝 뺍니다.
  3. 쌀뜬물로 하면 더 맛있다고 해요. 첫번째는 그냥 버리고 두번째 쌀뜬 물로 준비합니다.
  4. 참기름을 두르고 소고기를 볶습니다. 소고기라서 금방 익네요.
  5. 불린 미역을 물기를 빼서 넣고 3분정도 볶습니다.
  6. 냄비가 달라붙을거 같아서 코팅된 후라이팬에 볶았는데요. 볶은 고기와 미역을 냄비로 옮겨놓고 쌀뜬물을 넣습니다.
  7. 마늘 반술, 국간장 큰술 2를 넣습니다. 나머지 간은 꽃소금을 넣어가면서 잡습니다. 그리고 중불에 10분 약불에 20분정도 끓입니다.

4-5 내용 중 오디오 파일과 비디오 파일은 마땅히 삽입할만한 파일이 없어 이곳에 적겠습니다.

멀티 미디어 파일은 보통 object 태그와 embed 태그를씁니다. 형식은 object data="파일명" type=""[/object"]로 씁니다.


embed태그는[embed src="" type=""]를 쓰며 태그를 닫을 필요가 없습니다. object 태그는 data에 파일을 지정, embed 태그는 src에 파일을 지정합니다
위 두 코드는 재생 플러그인을 따로 요했으나, HTML5 이후로는 바로 삽입 재생이 가능해졌습니다.


그러므로 audio 와 video는 각각 [audio src=""][/audio] / [video src=""][/video] 태그로 가능하며 embed와 마찬가지로 src에 파일을 지정하면 됩니다.


파일 지정 방법은 image 삽입과 마찬가지로 같은 폴더 안에 하위 목록으로 audio, video를 만든 뒤 그대로 지정해주면 됩니다.
오디로를 배경으로 사용할거라면 audio src=""autoplay loof[/audio] 이렇게 autoplayloop를 넣어주면 됩니다. 비디오도 마찬가지입니다.
